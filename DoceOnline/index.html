ABOUT THE GAME

DOCE® Classic by Smart Iguana Games is a new two-player abstract strategy board game for all ages created by Luis & Dan Machado, father and son. 

Currently under development, this is the official video game version of DOCE® Classic. 

If you still dont know how to play, press the "Start" button on the window above and once you are in the game menu navigate to the option "HOW TO" for quick and easy-to-understand instructions about how to play DOCE. 

More info: https://smartiguanagames.com/doce/
